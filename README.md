🤖**Otto Labyrinth** <br/>
📌 *Overview*

- This project implements an obstacle-avoiding robot using Otto: https://hprobots.com. 
- The robot uses an ultrasonic sensor to detect obstacles and adjusts its movement in real time using a simple decision-making algorithm. 
- The goal of the project was to develop a basic navigation strategy that allows the robot to move forward when the path is clear and perform avoidance maneuvers when obstacles are detected.
